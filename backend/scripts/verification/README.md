# Verification Scripts

Scripts for verifying data quality and integrity.

## Scripts

- `verify_carrefour_data.py`: Verify Carrefour data quality
- `check_carrefour_images.py`: Check if product images are valid
- `count_qdrant_products.py`: Count products in Qdrant

## Usage

```bash
python scripts/verification/verify_carrefour_data.py
```
