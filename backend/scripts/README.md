# Scripts

Utility scripts for data management and operations.

## Folders

- **scraping/**: Scripts for scraping product data from retail websites
- **data_loading/**: Scripts for loading data into Qdrant vector database
- **verification/**: Scripts for verifying data quality
- **utilities/**: General utility scripts

## Usage

Run scripts from the backend directory:
```bash
cd backend
python scripts/scraping/scrape_carrefour_multi.py
```
