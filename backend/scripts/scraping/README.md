# Scraping Scripts

Scripts for scraping product data from retail websites.

## Scripts

- `scrape_carrefour_multi.py`: Scrape Carrefour products
- `scrape_carrefour_config.py`: Configuration for Carrefour scraper
- `load_retail_markets.py`: Load data from multiple retail markets

## Usage

```bash
python scripts/scraping/scrape_carrefour_multi.py
```
