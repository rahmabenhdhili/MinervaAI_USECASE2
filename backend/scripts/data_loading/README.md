# Data Loading Scripts

Scripts for loading scraped data into Qdrant vector database.

## Scripts

- `load_carrefour_to_qdrant.py`: Load Carrefour products to Qdrant
- `load_aziza_images_to_qdrant.py`: Load Aziza products with images
- `download_carrefour_images.py`: Download missing product images

## Usage

```bash
python scripts/data_loading/load_carrefour_to_qdrant.py
```
